Minimalist UI Prompt (Attendance System with QR & RFID)

Prompt:

Design a minimalist desktop application UI for an Employee Attendance System for SmartQ Systems.
The system is built with Python and connected to a Firebird database, and supports QR code scanning and RFID card scanning for recording attendance.

Theme

Primary color: #32AD32

Background: White

Style: Minimalist, clean, and professional

Simple icons and clear typography

Focus on fast interaction and easy navigation

Layout

Create a simple desktop-style interface.

Top Header

Company name: SmartQ Systems

Current date and time

User profile icon

Left Sidebar Navigation
Include only the essential features:

Time In / Time Out

Search Attendance

Generate Reports

Manual Logout

Main Screens
1. Time In / Time Out (Main Screen)

This is the primary screen for daily use.

Elements:

Large QR Code Scanner Area

RFID Scan Status indicator

Display text: “Scan QR Code or Tap RFID Card”

Show employee info after scan:

Employee Name

Employee ID

Department

Show recorded action:

Time In

Time Out

Success message after attendance is recorded.

2. Search Attendance

Elements:

Search bar (Employee Name or ID)

Date filter

Simple attendance table with:

Employee Name

Date

Time In

Time Out

Status

3. Generate Reports

Elements:

Date range selector

Department filter

Generate Report button

Attendance table preview

Export options

4. Manual Logout

Logout button

Confirmation popup:
“Are you sure you want to log out?”

UI Components

Green action buttons (#32AD32)

Simple tables

Minimal popups

Status indicators:

Green = Success

Red = Error

Yellow = Waiting for scan

Design Goal

Create a minimal, efficient attendance interface optimized for QR and RFID scanning, allowing employees to quickly record attendance with minimal steps.
